Q1: Які з описаних типів операційних систем призначені для роботи з обмеженими ресурсами та суворими вимогами щодо часу відгуку?

Багатокористувацькі операційні системи Системи з графічним інтерфейсом користувача Операційні системи реального часу Однокористувацькі операційні системи загального призначення

Description: Операційні системи реального часу розроблені для систем з обмеженими ресурсами, де критично важливо забезпечити гарантований час відгуку. Багатокористувацькі операційні системи призначені для одночасної роботи декількох користувачів, але не обов'язково з урахуванням реального часу. Системи з графічним інтерфейсом призначені для зручності користувача, але не гарантують реальний час відгуку. Однокористувацькі ОС загального призначення не адаптовані для суворих часових меж.
